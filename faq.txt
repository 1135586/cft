    Dow Jones Industrial Average index

    Фундаментальный анализ:
ВНП(GNP) // стоимость произведенного страной продукта и услуг как внутри страны, так и за рубежом. номинальный (в тек. ценах), реальная (с учетом инфляции)
Торговый баланс // данные по экспорту. увеличение прироста экспорта приводит к росту цен инструментов кредитного рынка, росту курса валют и росту курса акций
Платежный балансс // все коммерческие и финансовые сделки страны. увеличение говорит о том, что резиденты больше зарабатывают за границей, чем вывозят
Индекс цен производителя (Индекс оптовых цен)// среднее изменение цен, определяемых производителем для своей продукции (показатель будущего уровня инфляции)
CORE индекс // исключающий цены на энергию и продовольствие
Индекс цен поотребителей // ср. изменение рыночных цен корзины (отражает улисение инфляции)
    Учетыне ставки:
ODR(official discount rate) // ставка под которую ЦБ кредитует коммерчиские банки
Repo rate // ставка под которую ЦБ кредитует коммерческие банки под залог ценных бумаг(государственными казначейскими обязательствами)
Lombard Rate // ставка под которую ЦБ кредитует коммер. банки под залог недвидимости  или золотовалютных резервов
Unemployment Rate //  увеличение з/п -> инфляционное давление
Nonfarm Payrolls, Average Workweek, Average Hourly Earnings, Employment Cost Index
Денжная масса // наличные деньги в обращении и остатки безналичных средств у физ. юр. и государства
    агретаты:   M0 // наличные в обращении
                M1 // M0 + вклады до востребования
                M2 // M1 + срочные вклады
                M3 // M2 + сберегательные вклады, облигации
    (увеличение денежной массы ->  скачек инфляции, повышение кредитных ставок и падения цен на акции)

Industrial Production // общий выпуск продукции, объем комунальных услуг (сила национальной экономики)
Capacity Utilization // загруженность производственных мощьностей (81% оптимальное, выше - перегретое -> инфляция, ниже ослабление валюты)
NAMP (National Assiciation Of Purchaising Mangers) (как индекс деловой активности) // индекс по оптовым закупкам(опрос 4000 компаний) - влючает: заказы(30%), продукция(25%), занятость(20%), поставки(15%), запасы(10%)
                вычисляется взвешиванием для каждой категории (50% оптимально)
Factory Order // (потребность на заводскую продукцию)
Durable Goods Orders // (затраты на товары длительного полльзования) товары со сроком использования более 3 лет - индекс характеризует способность потребителей тратить большие суммы
Retail Sales, Consumer Confidence, Car Sales
Business Inventories // запасы на оптовых складах
Открытый интерес // количество позиций не закрытых на конец текущего дня
